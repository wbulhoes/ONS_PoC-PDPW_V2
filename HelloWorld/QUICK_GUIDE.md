```powershell

# Build
dotnet build

# Acessar manualmente
start http://localhost:5555

# Executar sem abrir navegador
# (comentar OpenBrowser no código)
```

## ?? Acessar de Outros Dispositivos

Para acessar de outro dispositivo na mesma rede:

1. Descubra seu IP:
```powershell
ipconfig
```

2. Acesse de outro dispositivo:
```
http://192.168.x.x:5555
```

## ?? Personalizações Fáceis

### Mudar Porta
Edite no `Program.cs`:
```csharp
var url = "http://localhost:5555"; // Mude aqui
```

### Desabilitar Auto-Open
Comente:
```csharp
// OpenBrowser(url);
```

### Mudar Cores
Edite o CSS no método `GenerateHtmlPage()`:
```css
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
```
