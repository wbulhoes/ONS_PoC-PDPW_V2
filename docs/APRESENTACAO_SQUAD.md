## ?? SLIDE 1: VISÃO GERAL

### **O que é a POC?**
Prova de Conceito para migração do sistema PDPW (Programação Diária de Produção) do stack legado para tecnologias modernas.

### **Stack Atual (Legado)**
- VB.NET + .NET Framework 4.x
- ASP.NET Web Forms
- SQL Server 2008
- Interface desktop

### **Stack Novo (Alvo)**
- C# 12 + .NET 8
- REST APIs + React 18
- SQL Server 2019+
- Interface web moderna

## ??? SLIDE 14: COMO EXECUTAR

### **Setup Rápido (5 minutos):**
```powershell
# 1. Clonar
git clone https://github.com/wbulhoes/ONS_PoC-PDPW_V2.git
cd ONS_PoC-PDPW_V2

# 2. Restaurar
dotnet restore

# 3. Migrations
dotnet ef database update --project src/PDPW.Infrastructure

# 4. Executar
dotnet run --project src/PDPW.API/PDPW.API.csproj

# 5. Acessar
# https://localhost:5001/swagger
```
