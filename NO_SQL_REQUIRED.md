# ?? Resumo: Você NÃO precisa instalar SQL Server!

## ? Resposta Rápida

**NÃO, você não precisa instalar SQL Server!**

A aplicação agora suporta **3 modos de banco de dados**:

---

## ?? Opções Disponíveis

### ?? Opção 1: InMemory Database (RECOMENDADO PARA COMEÇAR)

```
???????????????????????????????????????
?  ? Mais Rápido                     ?
?  ? Zero Instalação                ?
?  ? Funciona Imediatamente         ?
?  ?? Dados Temporários              ?
???????????????????????????????????????
```

**Configure em 10 segundos:**

1. Edite `src\PDPW.API\appsettings.Development.json`:
```json
{
  "UseInMemoryDatabase": true
}
```

2. Execute:
```powershell
dotnet run --project src\PDPW.API
```

**? PRONTO! A API está funcionando!**

?? **Guia completo:** `QUICK_START_INMEMORY.md`

---

### ? Opção 2: SQL Server LocalDB (MELHOR PARA DESENVOLVIMENTO)

```
???????????????????????????????????????
?  ?? Leve                            ?
?  ? Já vem com Visual Studio       ?
?  ? Dados Persistentes             ?
?  ? Suporta Migrations             ?
???????????????????????????????????????
```

**Verificar se já está instalado:**
```powershell
sqllocaldb info
```

Se já estiver instalado, configure:

```json
{
  "UseInMemoryDatabase": false,
  "ConnectionStrings": {
    "DefaultConnection": "Server=(localdb)\\mssqllocaldb;Database=PDPW_DB_Dev;Trusted_Connection=True;MultipleActiveResultSets=true"
  }
}
```

E crie o banco:
```powershell
dotnet ef database update --project src\PDPW.Infrastructure --startup-project src\PDPW.API
```

?? **Guia completo:** `DATABASE_SETUP.md` (Opção 2)
